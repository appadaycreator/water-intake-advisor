## 概要

**サービス名**: Water Intake Advisor
**バージョン**: 2.0.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/water-intake-advisor/

体重・年齢・性別・気温・運動量から1日に必要な水分量を正確に計算。脱水予防と健康管理に活用。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（キー: `waterAdvisor_v2`）
- **保存項目**: 体重・年齢・性別・気温・運動量
- **永続化**: ページ読み込み時に自動復元（restoreInputs関数）
- **クリア**: ブラウザのサイトデータ削除で初期化

## 計算ロジック（v2）

```
基本量 = 体重(kg) × 35(ml)
性別補正: 女性 × 0.92、男性/その他 × 1.0
年齢補正: 65歳以上 × 0.95、18歳未満 × 1.05
気温補正: 20〜24℃ +100ml / 25〜29℃ +200ml / 30〜34℃ +400ml / 35℃+ +600ml
運動補正: なし +0 / 軽め +300 / 中程度 +500 / 激しい +800
合計 = floor(基本量 × 性別/年齢補正 + 気温補正 + 運動補正)
```

## 使い方

1. 体重・年齢・性別を入力する
2. 今日の気温と運動量を入力・選択する
3. 「水分量を計算する」ボタンをクリックする
4. ゲージ・コップ数・時間帯別ガイドを確認する
5. 結果をコピー・シェアして活用する
